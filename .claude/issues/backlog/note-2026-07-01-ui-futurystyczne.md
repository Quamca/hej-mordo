# UI futurystyczne — wizualizacja głosu

Zamiast prostych słupków/kółek — bardziej klimatyczne okienko:
- Okrąg który pulsuje na zewnątrz gdy Gemini mówi
- Inna animacja (np. delikatne falowanie) gdy słucha
- Ogólny kierunek: futurystyczny, sci-fi feel

Kiedy: po zakończeniu EPIC-1 (Software Mordo PC) lub przy okazji GUI w późniejszej fazie.
Biblioteka do rozważenia: pygame, tkinter canvas animations, lub CustomTkinter.
